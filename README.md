# Movie Streaming Project

## Tech Stack
HTML · Tailwind CSS (CLI) · Vanilla JavaScript

## Team Setup — Do This After Cloning

### 1. Install dependencies
npm install

### 2. Run the Tailwind watcher (keep this terminal open while working)
npx tailwindcss -i ./src/styles/input.css -o ./src/styles/output.css --watch

### 3. Open with Live Server
Install the Live Server extension in VS Code.
Right-click index.html → Open with Live Server.
Never open HTML files by double-clicking — always use Live Server.
